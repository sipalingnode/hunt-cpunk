## Install Node.js
```
curl -fsSL https://deb.nodesource.com/setup_22.x | bash -
apt install -y nodejs
```
## Clone Repository
```
git clone https://github.com/sipalingnode/hunt-cpunk.git
cd hunt-cpunk
```
## Install Depency
```
npm init -y
npm install axios
```
